# YouTube Clone

A simple YouTube clone project created using HTML and CSS.  
This project replicates the basic UI design of YouTube including the navbar, sidebar, video thumbnails, and responsive layout.

## Features

- Responsive YouTube homepage design
- Navigation bar
- Sidebar menu
- Video thumbnail section
- Clean and modern UI
- HTML & CSS based project

## Technologies Used

- HTML5
- CSS3
